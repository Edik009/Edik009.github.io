# AASFA Scanner v5.0 Release Notes

## 🎉 Обновление v5.0: Универсальный Сканер Безопасности

### Что нового?

MAЯК (AASFA Scanner) превратился из инструмента для профессионалов в инструмент для всех!

#### 🌟 Интерактивное русское меню
Просто запусти `python main.py` и выбери что сканировать! Красивое меню с эмодзи и ASCII арт.

#### 📱 Универсальность
Теперь поддерживается 6+ типов устройств:
- Android-устройства
- Windows-компьютеры
- Mac
- iPhone/iPad
- Apple TV / Android TV
- Умные устройства (IoT)

#### 🔍 Авто-диагностика
Введи IP-адрес, и сканер сам определит тип устройства, покажет основную информацию и найденные проблемы.

#### 📊 Объяснения для новичков
Каждая уязвимость объясняется простыми словами:
- Что это?
- Где это?
- Что может произойти?
- Как это исправить?

#### 💾 Экспорт в 5+ форматах
- PDF (красивый отчёт)
- JSON (для программистов)
- HTML (для браузера)
- CSV (для Excel)
- TXT (простой текст)

#### 📜 История сканирований
Все сканирования сохраняются, можно посмотреть историю и статистику.

### Использование

#### Интерактивный режим (новичкам)
```bash
python main.py
```

Появится меню:
```
╔════════════════════════════════════════════════════════════════════╗
║       МАЯК - Универсальный Сканер Безопасности v5.0               ║
║            Проверка безопасности твоих устройств                  ║
╚════════════════════════════════════════════════════════════════════╝

Выбери что сканировать:
  1. 📱 Сканировать Android-устройство
  7. 🔍 Узнать что это за устройство (автоматическая диагностика)
```

#### CLI режим (профессионалам)
Все старые команды работают как раньше:
```bash
python main.py -t 192.168.1.100
python main.py -t 192.168.1.100 -m deep -o report.txt
```

    Что это?
    └─ На устройстве открыт старый сервис Telnet, который позволяет
       подключиться удалённо без какого-либо пароля.

    Где это?
    └─ Порт 23

    Что может произойти?
    └─ Злоумышленник может подключиться к устройству через telnet
       и выполнять любые команды (как администратор).

    Как это исправить?
    └─ 1. Отключи Telnet через меню настроек
       2. Если это возможно - обновись на новую версию Android
